## Tips:

- Gunakan format JPG atau PNG
- Kompres foto agar file size < 500KB per foto (gunakan tinypng.com)
- Resolusi recommended: 1080x1350 (portrait) atau 1080x1080 (square)
- Pilih foto yang berkualitas bagus dan tidak blur
