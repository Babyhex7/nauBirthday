# QRIS Code untuk Hadiah

Taruh QRIS code di sini dengan nama:

- qris-50k.png

## Cara Membuat QRIS:

1. **Buka aplikasi e-wallet** (GoPay, OVO, Dana, LinkAja, dll)
2. **Pilih menu "Terima"** atau "Request Money"
3. **Masukkan nominal: Rp 50.000**
4. **Generate QRIS code**
5. **Screenshot** QRIS code
6. **Save sebagai PNG** dengan nama `qris-50k.png`

## Tips:

- Format: PNG (agar background transparan)
- Pastikan QR code jelas dan tidak blur
- Crop dengan rapi, tidak ada bagian yang terpotong
- Test scan sebelum upload
- Resolusi recommended: 500x500 px atau lebih

## Alternative:

Jika mau taruh gambar lain (bukan QRIS), ganti nama file tetap jadi `qris-50k.png`
